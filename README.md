Prerequisites

Ensure you have JDK 8 or higher installed on your machine.

How to Run

Clone the repository:

git clone [https://github.com/Ahmed2015-22/ITI_Java.git](https://github.com/Ahmed2015-22/ITI_Java.git)


Navigate to the source folder:

cd ITI_Java/src


Compile and Run (example for PaintBrush):

javac paintbrush/Main.java
java paintbrush.Main
